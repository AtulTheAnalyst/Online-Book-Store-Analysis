📚 Online Book Store SQL Analysis
Project Overview

This project analyzes an online bookstore database using SQL. It includes database creation, table design, data exploration, and business-focused queries to generate meaningful insights from books, customers, and orders data.

Database Structure
Books – Book details, genre, price, and stock.
Customers – Customer information and location.
Orders – Purchase transactions and order details.
Key SQL Analysis
Book genre analysis
Customer segmentation
Revenue calculation
Best-selling books identification
Stock management insights
Customer spending analysis
Author-wise sales performance
Order trend analysis

SELECT Statements
WHERE Clauses
JOIN Operations

ORDER BY & LIMIT
Database Design & Relationships

To demonstrate SQL skills in database management, data analysis, query optimization, and business reporting using a real-world bookstore dataset.
